# Markdown Browser

A SwiftUI application for browsing and viewing markdown files.

## Features

- Browse markdown files
- View rendered markdown content
- Built with SwiftUI

## Requirements

- iOS 17.0+ / macOS 14.0+
- Xcode 15.0+
- Swift 5.9+

## Getting Started

1. Clone the repository
2. Open the project in Xcode
3. Build and run
